#❓ How to use it

```bash

git clone https://github.com/arthur-lage/e-commerce.git

cd e-commerce

yarn
# if you don't have yarn installed, you can use npm install. 
npm install

yarn start
# if you don't have yarn installed, you can use npm start. 
npm start

```
